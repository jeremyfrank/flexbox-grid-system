Flexbox Grid System

A simple, flexible and customizable mobile-first grid system based on CSS3 flexbox and custom html attributes with fractional width values.

View Project Source Share on Twitter

Custom Attributes

Instead of using classes, custom attributes and values are used for assigning layout and properties to a grid. Space-separated attribute selectors in CSS are used to set styling information. This approach is inspired by and similar to AMCSS.

Target Modifiers

Handle different layouts at different breakpoints with ease using target modifiers (lap & desk).

grid-item="fraction[:target]"
